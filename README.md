# Projet Fi Dounya Wal Ahyri

Ce projet est une application web qui utilise HTML, CSS et JavaScript pour offrir une interface utilisateur interactive. Il interagit également avec Supabase pour la gestion des données.

## Structure du projet

```
fi-dounya-wal-ahyri
├── index.html        # Structure de base de la page web
├── style.css         # Styles CSS pour la mise en forme
├── script.js         # Code JavaScript pour les fonctionnalités interactives
├── supabase.js       # Code pour interagir avec Supabase
├── vercel.json       # Configuration pour le déploiement sur Vercel
└── README.md         # Documentation du projet
```

## Installation

1. Clonez le dépôt :
   ```
   git clone <URL_DU_DEPOT>
   ```
2. Accédez au dossier du projet :
   ```
   cd fi-dounya-wal-ahyri
   ```

## Utilisation

- Ouvrez `index.html` dans un navigateur pour voir l'application en action.
- Modifiez `style.css` pour changer l'apparence de l'application.
- Ajoutez des fonctionnalités dans `script.js` pour améliorer l'interactivité.
- Utilisez `supabase.js` pour gérer les données et l'authentification avec Supabase.

## Déploiement

Pour déployer le projet sur Vercel, assurez-vous que le fichier `vercel.json` est correctement configuré. Ensuite, suivez les instructions de déploiement sur la plateforme Vercel.
